## Requirements

* A running Redis server
* jedis.jar >= 2.1.0 a Redis Java client.
* primefaces.jar >= 5.1 UI component library for JavaServer Faces (JSF).
* javax.faces-2.2.x.jar (JSF framework).
* A servlet container Apache Tomcat.
